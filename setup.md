Getting Atto up and running is designed to be simple. No complex environment configuration required.

# 1. Download Atto
Go to the Releases page and download the version appropriate for your operating system:

1. Windows: atto-windows.zip
2. macOS: atto-macos.zip
3. Linux: atto-linux.tar.gz

# 2. Unzip and Run
Extract the downloaded folder to a location of your choice.

1. Locate the Atto executable file inside the folder.
2. Double-click to run the application.

# 3. Configure your AI
Atto uses AI to parse your job listings. To enable this:

1. Obtain an API key from [OpenAI](https://platform.openai.com/settings/organization/api-keys). You may need to sign up for an account.
2. Open Atto and navigate to the Settings page.
3. Paste your API key into the designated field and click Save.
